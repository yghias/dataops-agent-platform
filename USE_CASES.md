# Use Cases

## Pipeline generation
Generate a Snowflake ingestion and transformation package with Airflow orchestration, quality checks, and operational documentation.

## Schema review
Review a proposed logical or physical schema for grain, keys, scalability, and downstream usability.

## Query optimization
Analyze a slow or expensive SQL query and recommend a rewrite plus physical design considerations.

## DAG generation
Generate an Airflow DAG with retries, SLAs, idempotency, partition handling, and failure notifications.

## KPI definition
Turn a business metric request into canonical Snowflake SQL and metric documentation with assumptions and edge-case handling.

## Runbook generation
Create an operator-friendly runbook for a recurring pipeline or data quality incident.

## ML workflow support
Design feature pipelines, validation logic, and productionization guidance for data science use cases.
