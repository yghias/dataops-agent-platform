# LinkedIn Summary

Built `dataops-agent-platform`, a production-style human-in-the-loop multi-agent AI platform for data engineering, architecture, cloud, database, analytics, and AI/ML workflows. The platform routes requests to specialist agents, uses constrained tools for schema, metadata, lineage, SQL, DAG, and dbt tasks, and requires human approval before any approved artifact moves toward execution.

The project demonstrates how to apply AI to real data platform work without sacrificing governance, reviewability, or operational control. It includes approval workflows, decision logging, feedback memory, observability design, SQL and documentation assets, CI/CD scaffolding, and portfolio-ready examples for ETL, query tuning, schema review, KPI definition, and runbook generation.

This project is a strong fit for conversations around Senior Data Engineer, Principal Data Architect, Senior Cloud Engineer, Senior Database Engineer, Senior DBA, Senior Data Analyst, Senior Data Scientist, and Senior AI/ML Engineer roles.
