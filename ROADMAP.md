# Roadmap

## Phase 1: Foundation
- finalize agent, tool, and workflow interfaces
- add request and artifact schemas
- document governance, observability, and operating model

## Phase 2: Integration
- connect SQL runner to a real warehouse
- integrate metadata and lineage sources
- add API endpoints and UI-friendly response shapes

## Phase 3: Control plane hardening
- policy-as-code enforcement
- RBAC and identity integration
- persistent workflow state store
- centralized telemetry

## Phase 4: Learning loop
- retrieval over approved artifacts
- prompt evaluation framework
- reviewer quality trend analysis
- route confidence calibration

## Phase 5: Enterprise readiness
- deployment targets
- environment promotion rules
- integration runbooks
- artifact signing and retention
