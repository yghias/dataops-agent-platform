# Project Summary

`dataops-agent-platform` is an enterprise-style repository for AI-assisted data platform operations. It combines Snowflake SQL models, Airflow orchestration, dbt-style transformations, governance controls, and specialist agents that generate reviewable artifacts for data engineering, architecture, database, analytics, and ML workflows.

The repository emphasizes SQL-first transformations and metrics, with Python reserved for orchestration, ingestion, tool execution, and workflow control. Generated artifacts are packaged with evidence and routed through human review before execution.
